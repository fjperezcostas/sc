# sc (Shell Calculator)
## Command line calculator
### Installation:
- Clone the repository.<br>
  `git clone https://github.com/fjperezcostas/sc.git`
- Move to **/sc** folder.<br>
  `cd sc`
- Run make tool.<br>
  `make`
- Run **sc** binary.<br>
  `./bin/sc`
### Example of use
```
sc v0.1
Written by F. Javier Perez - Type 'q' to exit
? 0
= 0
? 2+2
= 4
? 2+(-2)
= 0
? sqr4^2
= 4
? log(2-(-54.34))+sqr(2.9^(-.4))/2232.123-ln(12)
= -0.733728
? q

```
